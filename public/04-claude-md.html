Part 4: CLAUDE.md & Short Codes

Teaching Your AI

What is CLAUDE.md?

A file at your repo root that Claude reads automatically.

your-repo/
├── CLAUDE.md     <-- Claude reads this
├── src/
└── ψ/

Your AI's instruction manual.

CLAUDE.md Structure

# Project Name - AI Guide

## Golden Rules
(Things AI must NEVER do)

## Workflows
(How to do common tasks)

## Tools
(What tools to use)

## Patterns
(Project-specific patterns)

Golden Rules Example

## Golden Rules

1. NEVER use `--force` flags
2. NEVER push to main directly
3. NEVER merge PRs without approval
4. NEVER create temp files outside repo
5. NEVER use `git commit --amend`
6. Consult Oracle before debugging

Rules AI must follow. Non-negotiable.

Workflows Example

## Commit Workflow

1. Stage changes: `git add -A`
2. Check diff: `git diff --staged`
3. Write message: Focus on WHY not WHAT
4. Commit: `git commit -m "type: message"`
5. Push: `git push origin branch-name`

Never skip step 2!

Short Codes

Quick commands for common tasks.

Code Purpose
rrr Create retrospective
/recap Get session summary
/snapshot Quick knowledge capture
/distill Extract patterns

The rrr Command

Creates session retrospective.

# Session Retrospective

## AI Diary (Vulnerable Section)
What I struggled with, what surprised me...

## Work Done
- Task 1: completed
- Task 2: completed

## Learnings
- Pattern discovered: X

/recap Command

Fresh start context summary.

Used when:

  • Starting new session
  • Context is confused
  • After /jump to new topic
User: /recap
Claude: *reads recent retros, handoffs*
Claude: "Last session you were working on..."

/snapshot Command

Quick capture of current state.

User: /snapshot "Figured out the bug"
Claude: *creates timestamped file*

Creates: ψ/memory/logs/snapshots/2025-01-19_1430_figured-out-bug.md

Modular CLAUDE.md

For large projects, split into files:

CLAUDE.md           ← Main hub (lean)
CLAUDE_safety.md    ← Safety rules
CLAUDE_workflows.md ← Workflows
CLAUDE_subagents.md ← Agent docs
CLAUDE_lessons.md   ← Patterns

CLAUDE.md just links to others.

Skills Directory

Reusable instructions in ~/.claude/skills/

~/.claude/skills/
├── rrr/
│   └── SKILL.md
├── recap/
│   └── SKILL.md
└── snapshot/
    └── SKILL.md

Skills loaded on-demand. Saves context.
