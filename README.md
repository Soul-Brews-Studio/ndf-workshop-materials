## Slide Tech Stack

- **Reveal.js 5.0.4** (NO Marp)
- **Theme**: SIIT Dark (cyan/green accents)
- **Transitions**: None (instant)

## Quick Start

```bash

open public/01-slide-name.html

# Edit source
vim slides/01-slide-name.md
```
